## Skill trigger: `monitoring-check`

Invoke the `monitoring-check` skill when the user reports or asks about:
- postgres-exporter, query-exporter, or metric-collector not working
- Missing or stale Prometheus metrics for PostgreSQL
- ServiceMonitor or PrometheusRule resources
- Grafana dashboards for database monitoring
- PromQL queries for PostgreSQL metrics
- Alerting rules for database health
- Metrics scraping issues or targets down
- VictoriaMetrics or Prometheus configuration for database exporters
