## Skill trigger: `postgresql-backup-check`

Invoke the `postgresql-backup-check` skill when the user reports or asks about:
- Backup failures or backup not running
- pgBackRest status or backup history
- WAL archiving issues or archive_command failures
- Backup schedules or retention policies
- Point-in-time recovery (PITR) readiness
- Backup daemon pod status
- "When was the last successful backup?"
- Restore verification or backup integrity
