## Skill trigger: `postgresql-log-analyzer`

Invoke the `postgresql-log-analyzer` skill when the user reports or asks about:
- PostgreSQL or Patroni log errors
- FATAL errors in database logs
- OOM kills on database pods
- Deadlock detected in logs
- Operator reconciliation failures
- "What happened to the database at time X?"
- Timeline of events leading to an incident
- Unexpected database restarts or crashes
- Connection refused patterns in logs
