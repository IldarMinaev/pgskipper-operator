## Skill trigger: `postgresql-storage-check`

Invoke the `postgresql-storage-check` skill when the user reports or asks about:
- Disk full or "no space left on device" errors
- PVC capacity warnings or storage alerts
- Database or table size growth
- WAL accumulation or pg_wal directory growing
- Replication slot bloat (inactive slots retaining WAL)
- Table bloat estimation
- Index size or bloat
- Need to check how much disk space is used/available
