## Skill trigger: `pgskipper-check`

Invoke the `pgskipper-check` skill when the user reports or asks about:
- pgskipper-operator not working or in error state
- PostgreSQL CRD (Custom Resource) status issues
- Helm release problems for pgskipper-operator
- Operator deployment or pod issues
- Operator logs showing errors or reconciliation failures
- CRD not found or API version mismatch
- pgskipper-operator events in the namespace
- Cluster CR stuck in provisioning or error state
